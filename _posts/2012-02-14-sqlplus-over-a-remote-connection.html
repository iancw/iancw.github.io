To connect to a remote database with sqlplus without messing with TNS_PING or .ora config files, use this syntax:

sqlplus $USER@'(DESCRIPTION=(ADDRESS_LIST=(ADDRESS=(PROTOCOL=TCP)(HOST=$HOSTNAME)(PORT=1521)))(CONNECT_DATA=(SID=$DATABASE_NAME)))'

You can append a .sql file afterwards to have it run automatically.

sqlplus $USER@'(DESCRIPTION=(ADDRESS_LIST=(ADDRESS=(PROTOCOL=TCP)(HOST=$HOSTNAME)(PORT=1521)))(CONNECT_DATA=(SID=$DATABASE_NAME)))' @$SCRIPT_FILE
