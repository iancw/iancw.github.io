To enable the most critical emacs shortcuts, put this in ~/Library/KeyBindings/DefaultKeyBindings.dict

{
"~f" = "moveWordForward:";
"~b" = "moveWordBackward:";
"~d" = "deleteWordForward:";
}

You also probably want to turn on Option key as Meta in the Terminal preferences.
